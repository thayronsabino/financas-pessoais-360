## Por que preservar?

Estes documentos:
- ✅ Mostram a **lógica por trás** das decisões arquiteturais
- ✅ Servem como **referência** para futuros mantenedores
- ✅ Documentam o **caminho de evolução** (do que era para o que se tornou)
- ✅ Inspiram contribuidores a entender a **filosofia do projeto**
