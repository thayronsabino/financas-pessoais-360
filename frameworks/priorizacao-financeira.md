# Framework: Priorização Financeira

> **Tipo:** Framework de Decisão Financeira  
> **Versão:** 1.0.0  
> **Aplicação:** Decidir o que fazer primeiro quando há múltiplas frentes financeiras simultâneas  
> **Última atualização:** 2026-05-07

A maioria das pessoas não sofre por falta de plano — sofre por **excesso de frentes simultâneas**: quitar dívida, formar reserva, investir, contribuir, comprar casa, trocar carro, ajudar família, planejar aposentadoria. Este framework define a **ordem matematicamente correta** de prioridades, alinhada à mordomia cristã.

---

## 1. A Pirâmide de Priorização Financeira

```
              ┌─────────────────┐
              │   8. LEGADO     │  Patrimônio que sobrevive a você
              ├─────────────────┤
              │  7. EXPANSÃO    │  Múltiplas fontes de renda
              ├─────────────────┤
              │ 6. CRESCIMENTO  │  Investimentos de longo prazo
              ├─────────────────┤
              │ 5. OBJETIVOS    │  Casa, carro, educação dos filhos
              ├─────────────────┤
              │ 4. PROTEÇÃO     │  Reserva 6+ meses + seguros
              ├─────────────────┤
              │ 3. COLCHÃO      │  Reserva mínima R$ 1.500
              ├─────────────────┤
              │ 2. QUITAÇÃO     │  Dívidas com taxa > Selic
              ├─────────────────┤
              │ 1. ESSENCIAIS   │  Moradia, alimentação, saúde
              └─────────────────┘

REGRA: NÃO pular degraus. Cada nível só é abordado APÓS o anterior.
ÚNICA EXCEÇÃO: Dízimo (10%) é PRIMÍCIA — sai antes do nível 1.
```

## 2. Detalhamento de Cada Nível

### Nível 0 — DÍZIMO (Primícia, fora da hierarquia)

```
PRINCÍPIO: 10% sai antes de qualquer outra categoria
QUANDO: Sempre. Independe do nível em que está.
COMO: Transferência automática no dia do salário
```

### Nível 1 — Necessidades Essenciais

```
O QUE: Moradia, alimentação básica, transporte essencial, saúde mínima, educação
QUANTO DA RENDA: 45-60% (em estado SOBREVIVÊNCIA pode chegar a 70%)
ATÉ: Garantir que estas estão cobertas TODOS os meses
SE NÃO ESTÁ COBERTO: Não há como avançar para Nível 2 — voltar para
                     /pessoal-diagnostico-financeiro e identificar gargalo
```

### Nível 2 — Quitação de Dívidas Caras

```
O QUE: Eliminar TODAS as dívidas com taxa > Selic líquida (~12,5% a.a. em 2026)
ORDEM:
  Prioridade 1: Rotativo do cartão (~438% a.a.) ← URGÊNCIA MÁXIMA
  Prioridade 2: Cartão parcelado (~189% a.a.)
  Prioridade 3: Cheque especial (~136% a.a.)
  Prioridade 4: Crédito pessoal (~117% a.a.)
  Prioridade 5: CDC, financiamento veículo (~36-46% a.a.)
  Prioridade 6: Consignado público (~26% a.a., ainda > Selic líquida)
  
NÃO ENTRA AQUI: Financiamento imobiliário (taxa ~11% — discutir no Framework Casa vs. Aluguel)

REGRA: 100% do recurso livre concentrado nesta etapa, paralelamente apenas
       a um Colchão Mínimo (~R$ 100/mês para evitar nova dívida em emergência)
```

### Nível 3 — Colchão Mínimo (Camada 1 da Proteção)

```
O QUE: R$ 1.000-1.500 (1 mês de essenciais) em produto líquido
ONDE: Tesouro Selic, CDB liquidez diária
QUANDO: Construído paralelamente à quitação de dívidas caras
PARA QUÊ: Absorver pequenos imprevistos sem reiniciar ciclo de dívida
```

### Nível 4 — Camada de Proteção Financeira Completa

```
O QUE: 6-12 meses de gastos essenciais (varia conforme perfil profissional)
  CLT estável: 6 meses
  CLT instável: 6-9 meses
  Autônomo / freelancer: 9-12 meses
  PJ / empreendedor: 12 meses
  Renda variável: 12 meses

ONDE: Mix de Tesouro Selic + CDB liquidez diária + LCI/LCA pós-carência

INCLUI TAMBÉM SEGUROS BÁSICOS:
  + Seguro de vida (especialmente se há dependentes)
  + Plano de saúde (essencial)
  + Seguro residencial (se casa própria ou aluguel valioso)
  + Seguro automotivo (se há veículo financiado)

PARA QUÊ: Sustentar a família por período significativo em emergências reais
          (desemprego, doença grave, evento inesperado)
```

### Nível 5 — Objetivos Específicos

```
O QUE: Reservas para metas concretas com prazo definido
EXEMPLOS:
  + Casa própria (entrada + custos)
  + Troca de carro
  + Educação dos filhos (universidade, intercâmbio)
  + Casamento
  + Viagem significativa
  + Reforma estrutural

COMO ALOCAR:
  Curto prazo (< 3 anos):  100% renda fixa líquida (Tesouro Selic, CDB)
  Médio prazo (3-10 anos): Renda fixa + parte renda variável (Tesouro IPCA+, FIIs)

REGRA: Cada objetivo tem reserva separada (não misturar com Camada de Proteção)
```

### Nível 6 — Crescimento Patrimonial

```
O QUE: Investimentos de longo prazo (>10 anos), tipicamente para aposentadoria
ALOCAÇÃO TÍPICA (perfil moderado):
  20-30% Liquidez (Tesouro Selic, reserva tática)
  40-50% Renda Fixa Estruturada (Tesouro IPCA+, debêntures incentivadas)
  30-40% Renda Variável (ações, FIIs, fundos)

VEÍCULOS:
  + Tesouro RendA+ (específico para aposentadoria)
  + Previdência privada (PGBL para deduzir IR; VGBL no resto)
  + ETFs (BOVA11, IVVB11, fundos de índice)
  + Fundos imobiliários (FIIs)

REGRA: Disciplina mensal supera tentativa de timing de mercado
```

### Nível 7 — Expansão de Renda

```
O QUE: Diversificar fontes de renda para reduzir risco de fonte única
EXEMPLOS:
  + Renda passiva via dividendos crescentes
  + Imóveis para alugar
  + Trabalho de consultoria/freela complementar
  + Royalties (livros, cursos, conteúdo)
  + Empreendimento próprio em paralelo

QUANDO: Após Camada de Proteção completa + Investimentos de longo prazo ativos

REGRA: Expandir renda sem comprometer estabilidade já construída
```

### Nível 8 — Legado

```
O QUE: Patrimônio estruturado para impacto além da própria geração
COMPONENTES:
  + Testamento e planejamento sucessório
  + Holding familiar (se patrimônio justifica)
  + Fundos perpétuos para projetos ministeriais (endowments)
  + Mentoria e educação financeira da próxima geração
  + Apoio estrutural a ministérios

VEÍCULOS:
  + Doações em vida (inter vivos)
  + Disposições testamentárias
  + Estruturas jurídicas (holding, fundo patrimonial)
  + Seguro de vida estruturado para herança

REGRA: Mordomia plena (Provérbios 13:22) — pensar nos netos
```

## 3. Engine de Decisão: "O Que Fazer com Este Recurso?"

```
ENTRADA: Recurso livre disponível (R$ X)

PASSO 1 — Validar Nível 1 (Essenciais):
  SE há mês em déficit:
    → 100% para reorganizar essenciais
    → STOP — voltar para diagnóstico
  SE essenciais cobertos:
    → Avançar

PASSO 2 — Validar Nível 2 (Dívidas Caras):
  SE há rotativo ativo:
    → 100% do recurso (exceto R$ 100 para Colchão paralelo)
    → STOP até quitar
  SE há outras dívidas com taxa > Selic líquida:
    → 80-90% do recurso para quitação Avalanche
    → 10-20% para Colchão Mínimo
    → STOP até quitar todas

PASSO 3 — Validar Nível 3 (Colchão):
  SE Colchão Mínimo < R$ 1.500:
    → 100% para completar Colchão
    → STOP

PASSO 4 — Validar Nível 4 (Proteção Completa):
  SE Camada de Proteção < meta (6-12 meses):
    → 70-80% para completar Camada
    → 20-30% para iniciar objetivos específicos
    
  SE seguros básicos não estão contratados:
    → Contratar antes de avançar

PASSO 5 — Distribuir entre Níveis 5-6 (Objetivos + Crescimento):
  Conforme prioridades pessoais e prazos
  Tipicamente:
    Objetivos curto/médio prazo: 30-50% do recurso
    Crescimento longo prazo: 50-70% do recurso

PASSO 6 — Evoluir para Níveis 7-8 (Expansão + Legado):
  Apenas após anos consolidando os níveis anteriores
  Decisões de longo prazo, com aconselhamento especializado
```

## 4. Erros Comuns de Priorização

### Erro 1 — Investir antes de quitar dívida cara

```
SITUAÇÃO: Pessoa com R$ 5.000 no rotativo (438% a.a.) começa a "investir 
          R$ 200/mês em ações"

DIAGNÓSTICO: Erro grave de priorização
  Cada R$ 100 no rotativo gera R$ 36,5 de juros AO MÊS
  Cada R$ 100 em ações pode render R$ 1 a R$ 2 ao mês
  
CORREÇÃO: Pausar TODOS os investimentos. 100% para quitar rotativo.
```

### Erro 2 — Comprar imóvel sem reserva

```
SITUAÇÃO: Pessoa dá entrada de toda a reserva em apartamento financiado

DIAGNÓSTICO: Família vulnerável a qualquer choque
  Desemprego de 2 meses já compromete parcela
  Reparo emergencial força nova dívida
  
CORREÇÃO: Reconstruir Camada de Proteção como prioridade absoluta
          Antes de adquirir imóvel, ter Camada COMPLETA + entrada SEPARADA
```

### Erro 3 — Reserva grande demais

```
SITUAÇÃO: Pessoa com R$ 80.000 em reserva (10 meses) e nada em investimentos
          de longo prazo

DIAGNÓSTICO: Excesso de proteção, oportunidade perdida de crescimento
  Reserva ideal: 6 meses (R$ 48.000)
  R$ 32.000 ociosos rendendo Selic — abaixo do potencial de longo prazo
  
CORREÇÃO: Direcionar excedente para Tesouro IPCA+ ou outros instrumentos 
          de longo prazo
```

### Erro 4 — Trocar de carro com dívida ativa

```
SITUAÇÃO: Pessoa com cartão parcelado pendente troca de carro financiado

DIAGNÓSTICO: Acumulação de dívidas caras
  Nova parcela de carro + cartão = comprometimento crescente
  
CORREÇÃO: Quitar cartão antes de qualquer aquisição financiada
          Carro atual usado pode esperar
```

### Erro 5 — Adiar dízimo "até organizar"

```
SITUAÇÃO: "Quando eu pagar minhas dívidas, começo a dar o dízimo"

DIAGNÓSTICO: Inversão do princípio das primícias
  Dízimo é fundação, não topo
  
CORREÇÃO: Dízimo (10%) sai SEMPRE, em qualquer estado financeiro
          Plano de quitação inclui dízimo desde o primeiro mês
```

## 6. Adaptações por Fase de Vida

### Início de carreira (20-30 anos)
```
Foco principal: Níveis 1-4 (essenciais → proteção completa)
Acelerar: Construção de Camada de Proteção (juventude = mais flexibilidade)
Iniciar: Nível 6 (crescimento) com baixo aporte mas alta consistência
```

### Família com filhos pequenos (30-45 anos)
```
Foco principal: Níveis 4-5 (proteção + objetivos: educação dos filhos)
Especial atenção: Seguros (vida, residencial)
Crescimento: Aumentar aportes mensais para Nível 6
```

### Maturidade (45-60 anos)
```
Foco principal: Níveis 5-7 (objetivos + crescimento + expansão)
Especial atenção: Aposentadoria (Nível 6 acelerado)
Iniciar: Nível 8 (legado) com testamento e planejamento sucessório
```

### Pré-aposentadoria (60+ anos)
```
Foco principal: Níveis 6-8 (consolidação + expansão + legado)
Especial atenção: Realocar carteira (mais conservadora gradualmente)
Estruturar: Nível 8 (legado) com clareza
```

## 7. Princípios Operacionais

**Não pular degraus.** Tentar investir antes de quitar dívida cara é matemática contra você.

**Dízimo é primícia, não etapa.** 10% sai antes de qualquer um dos 8 níveis.

**Velocidade dos níveis varia.** Estado SOBREVIVÊNCIA pode levar 2 anos no Nível 2; estado EXPANSÃO progride rápido pelos níveis altos.

**Cada nível tem critério de saída claro.** Não se "gradua" do Nível 4 sem reserva completa. Sem ambiguidade.

**Recurso extra (13°, bônus, restituição) segue a pirâmide.** Direcionar para o nível atual de foco.

**Disciplina supera valor absoluto.** R$ 200/mês consistentes durante 30 anos derrotam R$ 5.000 esporádicos.

**Mordomia inclui ordem.** Provérbios 24:27 — "Prepara fora a tua obra, e apronta-a no campo, e depois edifica a tua casa." Construir na sequência correta é princípio bíblico de sabedoria.
