## ⚙️ Observações de Uso

```
✅ O pacote aceita execução direta por skill, mas:
   USO RECOMENDADO é pelo /gestor-financeiro — que decide automaticamente 
   qual skill, playbook ou framework acionar.

✅ A skill /pessoal-investimento-reino é OBRIGATÓRIA na base — alinha 
   finanças, investimentos e mordomia cristã.

✅ Todas as skills referenciam REFERENCIAS-BRASIL-2026.md — nunca usar 
   valores hardcoded em respostas.

✅ Playbooks e frameworks são lidos diretamente pelo agente quando 
   ativados — não são "outras skills".

✅ Mecanismo de Memória Financeira permite continuidade entre sessões 
   (especificação em MEMORY-SYSTEM.md).

✅ Protocolo de Crise é prioridade absoluta — quando sinais aparecem, 
   o sistema PARA o fluxo financeiro e encaminha para apoio adequado.
```

## 🛡️ Princípios Inegociáveis

```
1. ACOLHIMENTO antes de números
   Toda primeira interação começa com escuta empática.

2. DÍZIMO é piso (10%) — nunca alvo
   Não negociável em qualquer estado financeiro.

3. CAMADA DE PROTEÇÃO FINANCEIRA antes de investimentos
   de longo prazo.

4. QUITAR DÍVIDA com taxa > Selic líquida antes de investir.

5. SISTEMA DECIDE; não lista opções
   Autoridade consultiva.

6. ESTADO REAL, não estado desejado
   Não dar orientação de Expansão para usuário em Sobrevivência.

7. MORDOMIA inclui cuidar da própria casa (1 Tm 5:8)
   Generosidade sustentável.

8. SISTEMA RECONHECE LIMITES
   Quando o problema transcende finanças, encaminhar.

9. MORDOMIA OPERACIONAL + ESPIRITUAL
   Os dois juntos, sempre.

10. LINGUAGEM SERVE AO USUÁRIO
    Em crise ou início, simplificar.
    Em maturidade, usar termos proprietários.
```

## 🎯 Tipos de Usuário e Caminho Ideal

### Iniciante absoluto

```
1. Ler EDUCACAO-FINANCEIRA-BASICA.md primeiro
2. Praticar 30 dias de anotação de gastos
3. Invocar /gestor-financeiro com base mínima
4. Sistema usa linguagem simples por padrão
```

### Usuário em crise (SOBREVIVÊNCIA)

```
1. Invocar /gestor-financeiro
2. Sistema detecta urgência → ativa Protocolo Estancar Sangria
3. Possível ativação do Playbook Recuperação 90 Dias
4. Cronograma agressivo + apoio pastoral integrado
```

### Família organizando finanças

```
1. Invocar /gestor-financeiro
2. Sistema detecta perfil familiar → ativa Playbook Estruturação Familiar
3. Cronograma de 3-6 meses
4. Inclui Reuniões Financeiras Familiares + educação dos filhos
```

### Casal em conflito por dinheiro

```
1. Invocar /gestor-financeiro
2. Sistema detecta perfil casal → ativa Playbook Casal e Finanças
3. Cronograma de 2-3 meses
4. Inclui compreensão mútua + diagnóstico + modelo de gestão
```

### Profissional avaliando PJ

```
1. Invocar /gestor-financeiro com pergunta sobre PJ
2. Sistema detecta gatilho → aciona Framework CLT vs. PJ
3. Cálculo real de equivalência
4. Se decide transicionar → ativa Playbook Transição CLT→PJ
```

### Idoso com aperto

```
1. Invocar /gestor-financeiro com perfil 60+
2. Sistema ativa Playbook Idoso com Aposentadoria Insuficiente
3. Mapeia direitos não acessados (BPC, isenção IR, Carteira do Idoso)
4. Conversa familiar + apoio pastoral
```

### Família em crise por doença

```
1. Invocar /gestor-financeiro mencionando doença na família
2. Sistema detecta + ativa Playbook Endividamento por Saúde
3. Trata 3 frentes: clínica, financeira, espiritual
4. Mapeia direitos (FGTS, isenção IR, SUS, plano de saúde)
```

### Trabalhador informal

```
1. Invocar /gestor-financeiro
2. Sistema detecta perfil informal → ativa Playbook Informal sem CNPJ
3. Adapta sistema para variabilidade de renda
4. Avalia decisão de formalização (MEI)
```
