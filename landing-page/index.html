Inteligência Operacional

O orquestrador decide automaticamente.

Não é um chatbot. É um sistema com lógica de roteamento, classificação de urgência e seleção de especialistas.

01
Classificação de urgência

Analisa dívida ativa, reserva e fluxo de caixa para definir prioridade: baixa, média ou alta.

02
Roteamento automático

03
Checkpoints e limites

Evita sobrecarga cognitiva. Entrega o próximo passo, não um plano de 40 páginas.

Dúvidas Frequentes

Perguntas que você pode ter

Preciso ser desenvolvedor para usar?

Não. Se você usa Claude, Cursor, Windsurf ou qualquer agente com suporte a skills, basta instalar com um comando ou fazer upload manual dos arquivos SKILL.md. Não há código para escrever.

É realmente gratuito?

Sim. O pacote é open source sob Apache License 2.0 — uso pessoal e comercial permitido, sem mensalidades, sem planos pagos.

Preciso usar todos os 7 agentes?

Não. Você pode invocar qualquer agente diretamente. O /gestor-financeiro é o ponto de entrada recomendado e decide automaticamente qual caminho seguir com base na sua situação.

Como funciona a parte cristã?

Cada agente integra dízimo, ofertas e investimento no Reino como categorias orçamentárias reais — não como sugestões opcionais. Os princípios de Salmo 24:1, Lucas 16:10 e Provérbios 22:7 fundamentam cada recomendação.

Meus dados financeiros ficam seguros?

Os agentes rodam no contexto da sua conversa com a IA. Nenhum dado é enviado a servidores externos do pacote — tudo fica dentro da plataforma que você já usa (Claude, Cursor, etc.).

Posso contribuir ou adaptar o pacote?

Sim. O repositório é aberto no GitHub sob Apache License 2.0. Você pode fazer fork, adaptar e redistribuir — inclusive para uso comercial, desde que referencie a fonte original: github.com/thayronsabino/mordomia
