# Pessoal - Rotina Financeira Mensal

## Visão Geral

Este skill transforma intenções financeiras em disciplina mensal consistente. O problema não é falta de planejamento — é falta de revisão sistemática. Esta rotina fecha o ciclo: planejar → executar → revisar → ajustar.

A rotina cobre três pilares:
1. **Fechamento contábil** — O que realmente aconteceu vs. o planejado
2. **Análise de desvios** — Por que categorias estouraram e como corrigir
3. **Recalibração** — Ajustar metas, aportes e contribuições para o próximo ciclo

## Estrutura da Rotina Mensal

### Fase 1: Fechamento (Dias 1-5 do mês)

**Checklist de coleta:**
- [ ] Consolidar extratos de todas as contas correntes
- [ ] Consolidar faturas de cartões de crédito
- [ ] Registrar investimentos realizados (aportes, resgates, rendimentos)
- [ ] Registrar contribuições do reino (dízimo, ofertas, doações)
- [ ] Anotar receitas extras ou variações de renda

**Output esperado:** Tabela consolidada com todas as transações categorizadas.

### Fase 2: Análise Comparativa (Dias 6-10)

Construir tabela de comparação:

| Categoria | Planejado | Realizado | Desvio | Desvio % |
|-----------|-----------|-----------|--------|----------|
| Moradia | R$ X | R$ Y | R$ Z | +/-% |
| Alimentação | ... | ... | ... | ... |
| Transporte | ... | ... | ... | ... |
| Investimentos | ... | ... | ... | ... |
| Reino | ... | ... | ... | ... |

**Identificar padrões:**
- Categorias consistentemente estouradas (3+ meses) indicam orçamento irreal — ajustar para cima e compensar em outra categoria
- Categorias consistentemente abaixo do planejado podem liberar margem para investimentos ou reino
- Desvios pontuais (1 mês) requerem apenas atenção no próximo ciclo

### Fase 3: Diagnóstico de Desvios

Para cada categoria com desvio > 15%, responder:
1. **Foi pontual ou recorrente?** (Checar últimos 3 meses)
2. **Causa raiz?** (Emergência, impulso, necessidade real, orçamento subdimensionado)
3. **Ação corretiva?** (Ajustar orçamento, eliminar gasto, substituir por alternativa mais barata)

**Exemplo de diagnóstico:**
> "Alimentação estourou 28% em março. Analisando os últimos 3 meses: fevereiro +12%, janeiro +8%. Causa: aumento de pedidos por aplicativo (3x/semana). Ação: reduzir para 1x/semana, preparar marmitas. Economia esperada: R$ 200/mês."

### Fase 4: Recalibração de Metas (Dias 11-15)

**Revisar metas trimestrais:**
- Qual era a meta? (Ex: "Formar reserva de emergência de R$ 3.000")
- Progresso atual? (Ex: "R$ 1.200 acumulados, faltam R$ 1.800")
- Está no ritmo? (Ex: "Precisava de R$ 1.500 neste ponto — atrasado")
- Ajuste necessário? (Ex: "Aumentar aporte mensal de R$ 300 para R$ 450")

**Revisar contribuições do reino:**
- Dízimo está sendo calculado sobre renda bruta ou líquida? Manter consistência.
- Ofertas estão alinhadas com capacidade atual? Se orçamento apertou, revisar sem culpa — generosidade sustentável é melhor que insustentável.
- Há oportunidades de aumentar contribuição? Se sobraram recursos, considerar aumento.

### Fase 5: Planejamento do Próximo Ciclo (Dias 16-20)

**Gerar novo orçamento ajustado:**
1. Partir do orçamento anterior
2. Aplicar ajustes identificados na análise
3. Incorporar eventos pontuais do próximo mês (aniversários, viagens, manutenções)
4. Definir meta de investimento realista
5. Confirmar contribuições do reino

**Definir 1-3 micro-hábitos para o mês:**
Em vez de metas genéricas ("gastar menos"), criar hábitos específicos:
- "Registrar todo gasto acima de R$ 50 no mesmo dia"
- "Revisar saldo toda sexta-feira às 18h"
- "Preparar almoço 4x/semana"

Micro-hábitos criam consistência. Metas grandes dependem de micro-hábitos diários.

## Template de Output

Ao executar esta rotina, gerar:

```
# FECHAMENTO FINANCEIRO — [MÊS/ANO]

## 1. RESUMO EXECUTIVO
- Receita total: R$ X
- Despesas totais: R$ Y
- Saldo do mês: R$ Z
- Taxa de poupança: W% (investimentos + reserva / receita)

## 2. DESVIOS IDENTIFICADOS
[Listar categorias com desvio > 10%, com causa e ação corretiva]

## 3. PROGRESSO DE METAS
[Para cada meta trimestral: status, % de progresso, se está no ritmo]

## 4. CONTRIBUIÇÕES DO REINO
- Dízimo: R$ X (Y% da renda)
- Ofertas/doações: R$ Z
- Total reino: R$ W
- Reflexão: [Está alinhado com valores e capacidade?]

## 5. ORÇAMENTO AJUSTADO PARA [PRÓXIMO MÊS]
[Tabela com categorias e valores ajustados]

## 6. MICRO-HÁBITOS DO MÊS
1. [Hábito específico 1]
2. [Hábito específico 2]
3. [Hábito específico 3]

## 7. PRÓXIMA REVISÃO
Agendar para: [Data entre dias 1-5 do próximo mês]
```

## Princípios de Execução

**Consistência > Perfeição:** Melhor fazer revisão de 80% de qualidade todo mês do que revisão perfeita a cada 3 meses. O hábito é mais valioso que a precisão absoluta.

**Dados, não emoções:** Decisões financeiras baseadas em "acho que gastei muito" falham. Decisões baseadas em "gastei 28% acima em alimentação" funcionam.

**Ajustar sem culpa:** Orçamento estourado não é falha moral — é feedback. Ajuste e siga em frente. Culpa paralisa, dados orientam.

**Generosidade sustentável:** Contribuições do reino devem ser generosas E sustentáveis. Se a situação apertou, ajustar temporariamente é sabedoria, não falta de fé.

## Integração com Outras Skills

Esta rotina é o **hub central** do sistema financeiro pessoal:
- Use `/pessoal-diagnostico-financeiro` para diagnóstico inicial antes de começar a rotina
- Use `/pessoal-orcamento-domestico` para criar o orçamento base que será revisado mensalmente
- Use `/pessoal-estrategia-investimentos` para definir onde alocar excedentes identificados na revisão
- Use `/pessoal-investimento-reino` para aprofundar reflexão sobre contribuições e generosidade
- Use `/gestor-financeiro` para visão estratégica trimestral/anual

A rotina mensal mantém o sistema vivo. Sem ela, planejamento vira documento morto.

## Quando NÃO Usar Esta Skill

- **Primeiro contato com finanças pessoais:** Comece com `/pessoal-diagnostico-financeiro` para entender situação atual
- **Definir estratégia de investimentos:** Use `/pessoal-estrategia-investimentos`
- **Criar orçamento do zero:** Use `/pessoal-orcamento-domestico`
- **Revisão anual ou trimestral estratégica:** Use `/gestor-financeiro`

Esta skill é para **manutenção mensal de sistema já implantado**. Se o sistema não existe, construa-o primeiro.
